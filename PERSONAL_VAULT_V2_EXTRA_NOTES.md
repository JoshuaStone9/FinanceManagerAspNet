# Personal Vault v2 extra changes

Added after the integrated v2 redesign:

## Personal Vault
- Added `BeingDelivered` item status.
- Added optional `CustomStatus` so you can type your own visible status label.
- Added `ItemPhotos` table/entity for extra item pictures.
- Added additional photos upload on create/edit and item detail pages.
- Added photo deletion from item detail.
- Added "Duplicate as template" action from item detail to create another item of the same kind.

## Finance Manager
- Added stocks + crypto tracking separate from ISA/pots.
- Added stocks + crypto update form on Dashboard and Statistics.
- Added top-level Total Value calculations:
  - current total = goal accounts + Personal Vault value + stocks/crypto.
  - April total = goal forecast by April + Personal Vault value + stocks/crypto forecast.
- Added account editing/deleting/resetting from Statistics and Dashboard update modal.
- Added delete actions for bills, extra expenses, investments and savings rows.
